# MyPortfolio 🌐✨

A modern and responsive personal developer portfolio website built using **HTML, CSS, and JavaScript** to showcase my projects, skills, blogs, and contact information.

---

## 📌 About The Project

This portfolio website is designed to represent my journey as a developer and display my recent work in a clean and interactive way.

The website includes:

* About Me section
* Featured Projects
* Skill Stack
* Blogs & Articles
* Contact section
* Responsive sidebar navigation
